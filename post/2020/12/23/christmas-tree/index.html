初始化

新建页面，添加一个 canvas 元素，引入 css, js 文件

画树枝

使用 stroke 方法绘制树枝，设置旋转角度绘制左右子树，保存状态，递归绘制子树。

画树叶

获取画布所有像素点 alpha 通道值，判断此处是否有图像，循环像素点数组绘制半圆。

画礼物

使用 fillText 和 drawImage 方法绘制文字和图片。

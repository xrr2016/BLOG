第一题

JavaScript 采用原型继承，即一个对象继承自另外一个对象，另外一个对象再继承自别的对象，依此往复。请写一个通用的 JavaScript 函数，来找出某个对象身上的某个属性继承自哪个对象。

解答: 实际上是在考察如何判断一个对象上的属性是本身的还是通过继承来的，使用 hasOwnProperty 和 getPrototypeOf 方法即可
